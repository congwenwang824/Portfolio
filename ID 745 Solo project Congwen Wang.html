Dataset: Heart failure clinical records Data Set

Numerical variables: age, anaemia, creatinine_phosphokinase, diabetes, ejection_fraction, high_blood_pressure, platelets, serum_creatinine, serum_sodium, sex, smoking, time, and DEATH_EVENT

Plan to do: classification.

Class (y): Death Event (Death=1, Survive=0)

Describtion: "This dataset contains the medical records of 299 patients who had heart failure, collected during their follow-up period, where each patient profile has 13 clinical features." (Chicco and Jurman, 2020).

Classification methods: K-nears Neighbor, Linear Discriminart, Quadratic Discriminant Anlysis, Logistic regression Classification, Decision Tree.

Project purpose: Find out the best classification method for this Heart failure clinical records dataset.
